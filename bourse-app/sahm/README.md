# پوشه sahm

هر فایل CSV که در این پوشه قرار می‌دهید باید داده‌ی تاریخی یک یا چند نماد را
با ستون‌های دقیقاً زیر داشته باشد (ردیف اول = هدر):

```
Ticker,DTYYYYMMDD,FIRST,HIGH,LOW,CLOSE,VALUE,VOL,OPENINT,PER,OPEN,LAST
Kahroba,20260720,172888,174299,171900,173016,4015720000000,23210165,12517,D,178735,172900
```

## نکات

- می‌توانید هر تعداد فایل CSV در این پوشه بریزید — یک فایل برای هر نماد،
  یا چند نماد در یک فایل. برنامه تمام ردیف‌ها را می‌خواند و بر اساس ستون
  `Ticker` گروه‌بندی می‌کند.
- اگر یک نماد در چند فایل پخش شده باشد، برنامه همه را با هم ادغام و
  بر اساس تاریخ مرتب می‌کند.
- نام فایل اهمیتی ندارد؛ فقط پسوند `.csv` باید باشد.
- برای بارگذاری، داخل برنامه روی دکمه‌ی «وارد کردن پوشه sahm» بزنید و
  همین پوشه را انتخاب کنید. با هر تغییر در فایل‌های این پوشه (افزودن/ویرایش)،
  دوباره همان دکمه را بزنید تا داده‌ها به‌روز شوند.
- داده‌ها پس از وارد شدن در حافظه‌ی محلی مرورگر (IndexedDB) ذخیره می‌شوند،
  پس در بازدیدهای بعدی نیازی به وارد کردن دوباره نیست — مگر بخواهید بروزرسانی کنید.
